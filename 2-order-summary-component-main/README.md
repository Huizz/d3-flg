### What I learned

- Use CSS reset to remove browser stylesheet styles
```
body,
main,
h1,
p,
div,
a,
button {
  margin: 0;
  padding: 0;
  border: 0;
  font-size: 100%;
  font: inherit;
  vertical-align: baseline;
  font-family: inherit;
}
```
- Use landmarks
```
<main>...</main>
<footer>...<footer>
```
- Using box-shadow to create shadow effect for elements
